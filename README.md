# cojira

Agent-first CLI toolkit for Confluence and Jira automation.

## Install

Preferred: use the local release bundle in the workspace and follow `COJIRA-BOOTSTRAP.md`.

Alternative (requires Go 1.22+):

```bash
go build -o cojira .
```

## Setup

```bash
cp .env.example .env
```

See [CLAUDE.md](CLAUDE.md) for full documentation.
